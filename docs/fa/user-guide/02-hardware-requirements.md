# فصل ۲ — سخت‌افزار مورد نیاز

## هدف این فصل

پیش از شروع نصب نرم‌افزارها باید مطمئن شویم سخت‌افزار مناسبی در اختیار داریم.

در این فصل بررسی می‌کنیم:

- چه مدل Raspberry Pi مناسب است؟
- چه نوع کارت حافظه‌ای استفاده کنیم؟
- چه منبع تغذیه‌ای مناسب است؟
- آیا Wi-Fi کافی است یا بهتر است از کابل شبکه استفاده کنیم؟
- حداقل و حداکثر سخت‌افزار پیشنهادی چیست؟

در پایان این فصل می‌توانید با اطمینان تصمیم بگیرید که آیا تجهیزات فعلی شما برای اجرای CCC مناسب هستند یا خیر.

## 2.1 Raspberry Pi چیست؟

**هدف**

آشنایی با دستگاهی که قرار است CCC و Conduit روی آن اجرا شوند.

**چرا؟**

Conduit و CCC می‌توانند روی انواع مختلفی از سیستم‌های لینوکسی اجرا شوند، اما Raspberry Pi یکی از محبوب‌ترین گزینه‌ها است زیرا:

- مصرف برق بسیار کمی دارد.
- قیمت آن نسبتاً مناسب است.
- به صورت 24 ساعته قابل استفاده است.
- جامعه کاربری بزرگی دارد.
- مستندات فراوانی برای آن وجود دارد.

می‌توان Raspberry Pi را یک رایانه کامل کوچک در نظر گرفت که اندازه آن تقریباً به اندازه یک کارت بانکی است.

## 2.2 کدام مدل Raspberry Pi مناسب است؟

**پیکربندی‌های اعتبارسنجی‌شده**

- Raspberry Pi 4 (2 GB RAM)
- Raspberry Pi 3 Model B (1 GB RAM)

CCC نسخهٔ v0.3.1 روی این پیکربندی‌ها با موفقیت نصب، پیکربندی و اجرا شده است (سند اعتبارسنجی در `docs/health-checks/`).

**پیکربندی پیشنهادی**

Raspberry Pi 4 (4 GB RAM)

**گزینه‌های دیگر**

Raspberry Pi 4 با 8GB RAM

اگر در دسترس دارید انتخاب بسیار خوبی است، اما برای اجرای CCC الزام محسوب نمی‌شود.

**Raspberry Pi 5**

در زمان نگارش این مستند Raspberry Pi 5 نیز گزینه مناسبی است و از نظر عملکرد از Raspberry Pi 4 سریع‌تر است.

## 2.3 کارت حافظه (MicroSD)

**هدف**

انتخاب حافظه‌ای که سیستم عامل روی آن نصب می‌شود.

**چرا اهمیت دارد؟**

تمام سیستم عامل، تنظیمات و فایل‌های مورد نیاز روی کارت حافظه ذخیره می‌شوند.

کارت حافظه بی‌کیفیت می‌تواند باعث:

- کندی سیستم
- خرابی فایل‌ها
- از دست رفتن اطلاعات
- خرابی سیستم عامل

شود.

**حداقل پیشنهادی**

32GB

**پیشنهاد اصلی**

64GB

**برندهای پیشنهادی**

- Samsung
- SanDisk
- Kingston

ترجیحاً از مدل‌های شناخته‌شده و اصلی استفاده کنید.

## 2.4 منبع تغذیه

**هدف**

اطمینان از تأمین برق پایدار برای Raspberry Pi.

**چرا اهمیت دارد؟**

بسیاری از مشکلاتی که کاربران به اشتباه به نرم‌افزار نسبت می‌دهند در واقع ناشی از منبع تغذیه نامناسب هستند.

منبع تغذیه ضعیف ممکن است باعث:

- ریبوت ناگهانی
- ناپایداری سیستم
- خرابی کارت حافظه
- اختلال در عملکرد شبکه

شود.

**پیشنهاد**

استفاده از آداپتور رسمی Raspberry Pi یا یک منبع تغذیه با کیفیت و توان کافی.

## 2.5 اتصال شبکه

**هدف**

انتخاب روش اتصال Raspberry Pi به شبکه خانگی.

**گزینه اول: Wi-Fi**

مزایا:

- نیاز به کابل ندارد.
- راه‌اندازی ساده‌تر است.

معایب:

- پایداری کمتر
- حساسیت به فاصله و موانع
- تأخیر بیشتر

**گزینه دوم: Ethernet (کابل شبکه)**

مزایا:

- پایداری بیشتر
- سرعت بهتر
- تأخیر کمتر
- مناسب برای کارکرد 24 ساعته

**پیشنهاد**

در صورت امکان از اتصال کابلی استفاده کنید.

برای اجرای یک گره دائماً فعال، Ethernet انتخاب بهتری است.

## 2.6 تجهیزات جانبی مورد نیاز

برای راه‌اندازی اولیه ممکن است به موارد زیر نیاز داشته باشید:

- کابل شبکه
- کارت حافظه MicroSD
- کارت‌خوان MicroSD
- منبع تغذیه مناسب
- رایانه شخصی برای آماده‌سازی کارت حافظه

## 2.7 تجهیزات اختیاری

این تجهیزات الزامی نیستند اما می‌توانند مفید باشند.

**قاب محافظ (Case)**

محافظت فیزیکی بهتر.

**هیت‌سینک**

کمک به خنک‌سازی سیستم.

**فن**

برای محیط‌های گرم یا استفاده طولانی‌مدت.

**UPS**

برای جلوگیری از خاموش شدن ناگهانی هنگام قطع برق.

## 2.8 پیکربندی پیشنهادی

اگر قصد دارید یک گره پایدار و کم‌دردسر داشته باشید، ترکیب زیر پیشنهاد می‌شود:

- Raspberry Pi 4 (4GB RAM)
- MicroSD 64GB
- Ethernet
- منبع تغذیه با کیفیت
- قاب محافظ

این ترکیب برای اکثر کاربران کاملاً کافی است.

**اعتبارسنجی این فصل**

پیش از ادامه باید بتوانید به پرسش‌های زیر پاسخ دهید:

1. از چه مدل Raspberry Pi استفاده خواهید کرد؟
2. چه ظرفیتی برای کارت حافظه انتخاب کرده‌اید؟
3. آیا از Wi-Fi استفاده می‌کنید یا Ethernet؟
4. آیا منبع تغذیه شما مناسب است؟
5. آیا تمام تجهیزات لازم را تهیه کرده‌اید؟

اگر پاسخ این پرسش‌ها مشخص است، آماده ورود به فصل بعد هستید.

**فصل بعد**

در فصل بعد سیستم عامل Ubuntu را روی Raspberry Pi نصب خواهیم کرد و اولین راه‌اندازی دستگاه را انجام می‌دهیم.
